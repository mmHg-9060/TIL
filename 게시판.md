# 게시판을 만들 때 사용했던 기술

## 언어 및 뼈대
기술: Java, Spring Boot
역할: Java(메인 언어), Spring Boot(프레임워크)

## 웹 요청 처리
기술: Spring MVC
특징: @Controller, @GetMapping을 이용해서 브라우저의 요청을 받은 다음, 응답 흐름 제어

## DB 및 ORM
기술: Spring Data JPA, Hibernate
특징: 복잡한 SQL 쿼리를 직접 작성하지 않고, Entity를 통해 DB를 조작하게 해줌

## 뷰 템플릿 엔진
기술: Mustache
특징: 서버에서 넘겨준 데이터를 HTML에 채워넣고, 헤더와 푸터 등 화면을 조립함.

## 프론트엔드
기술: Bootstrap
특징: btn-primary, form-control을 사용해서 반응형 화면 디자인을 구현함.

## MVC
### Model
데이터와 상태를 관리

### View
사용자에게 보여지는 화면

### Controller
사용자의 요청을 받아 Model에게 넘기고, 결과를 View에게 넘겨줌

## DTO
화면에서 입력받는 데이터(AtricleFrom DTO)와 실제 DB 테이블을 원본 데이터(Article Entity)를 나눠줌
